# EHX 下线防错程序 - 操作员使用说明

## 启动

双击 `EHX下线防错.exe`。程序自动全屏，并在右上角显示当前扫码进度。

## 扫码

1. 确认界面状态为“待扫码”。
2. 使用扫码枪逐件扫描，扫码枪回车会自动触发校验。
3. 成功扫码后，已扫数量和右上角大号进度立即增加。
4. 重复、混料、格式错误或未配置物料会弹出红色错误提示，进度不会增加。
5. 满箱后程序自动生成下线单；Windows 正式模式会自动发送打印。
6. 新箱开始后进度自动恢复为 `0/每箱数量`。

## 历史查询

点击“历史查询 / 补打”，可以按完整条码、下线单号或日期查询。选择记录后可以
打开 PDF；Windows 正式环境可以执行补打。

## 异常处理

- PDF生成失败或打印失败时，不要重复扫描新箱。
- 点击“重试满箱处理”，或联系管理员检查环境。
- 数据和已生成 PDF 会保留，不会因打印失败丢失。

## 退出

按 `Esc` 或 `Ctrl+Q` 退出程序。
