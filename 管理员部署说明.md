# EHX 下线防错程序 - 管理员部署说明

## 环境要求

- Windows 10/11 64 位。
- Microsoft Excel 已安装并能正常启动。
- 现场打印机驱动已安装，Windows 测试页打印正常。
- 程序目录对当前用户可写。

程序默认使用代码生成的 Code128 PNG，不需要安装条码字体，也不需要安装额外的
PDF 转换或静默打印软件。

## 部署

1. 解压 `EHX下线防错_Windows.zip`。
2. 保持 EXE、`config.json`、物料表和模板位于同一目录。
3. 在 `config.json` 中设置 `printer_name`；留空则使用系统默认打印机。
4. 确认 `box_scan_count`、产线名称和工位名称正确。
5. 运行 `EHX下线防错.exe`。

## 必需的同级文件

- `EHX下线防错.exe`
- `config.json`
- `EHX物料号匹配.xlsx`
- `报交下线单模板.xlsx`

## 验收

1. 执行正常、重复、混料、未配置和格式错误扫码测试。
2. 扫满一箱，确认生成单页 A5 横向 PDF。
3. 检查纸面物料号、客户料号和下线单号条码可扫描。
4. 模拟打印失败，确认 PDF、XLSX 和数据库记录仍保留。
5. 在历史查询中打开 PDF 并执行补打。

## 数据目录

- `data/`：SQLite 数据库。
- `logs/`：运行与错误日志。
- `output/pdf/`：PDF 和同名 XLSX 留档。
- `output/barcodes/`：程序生成的 Code128 PNG。

程序直接使用 SQLite，不依赖外部数据库管理工具。
